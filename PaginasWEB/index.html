Comandos de Manutenção do Sistema

DISM /Online /Cleanup-Image /CheckHealth

Verifica se há alguma corrupção no sistema que possa impedir o funcionamento correto do Windows.

DISM /Online /Cleanup-Image /ScanHealth

Realiza uma verificação mais profunda para detectar se há alguma corrupção no sistema.

DISM /Online /Cleanup-Image /RestoreHealth

Repara automaticamente qualquer corrupção encontrada no sistema.

SFC /Scannow

Verifica todos os arquivos protegidos do sistema e substitui os arquivos corrompidos por uma cópia em cache localizada em uma pasta comprimida em %WinDir%\System32\dllcache.

MRT

Executa a Ferramenta de Remoção de Software Mal-Intencionado do Windows para procurar e remover malware.

chkdsk /f /r /x

Verifica o disco rígido em busca de erros e tenta corrigir quaisquer problemas encontrados. O parâmetro /f corrige erros no disco, /r localiza setores defeituosos e recupera informações legíveis, e /x força o volume a ser desmontado antes da verificação.

DISM /Online /Cleanup-Image /RestoreHealth /Source:X:\Sources\install.wim

Repara a imagem do Windows usando um arquivo de imagem específico (install.wim) localizado no caminho fornecido.
